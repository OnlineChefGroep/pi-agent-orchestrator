## What is this?

A [Pi extension](https://github.com/OnlineChefGroep) that adds autonomous
sub-agent orchestration to a Pi coding agent. Spawn specialized agents
(Explore, Plan, Analysis, general-purpose, or your own), chain them through
structured handoffs, schedule recurring jobs, and coordinate multi-agent
swarms — all from a vim-style interactive TUI dashboard.

The extension is **local-only**: it runs in your Pi host process, makes no
outbound network calls of its own, and stores no user data on a server.

## Installation

**Prerequisites:** Node.js 22 or newer and a working Pi host with
`@earendil-works/pi-coding-agent` 0.80.6 or newer.

Install the extension globally into your Pi environment:

```bash
pi install npm:@onlinechefgroep/pi-agent-orchestrator
```

Or install it for the current project only:

```bash
pi install npm:@onlinechefgroep/pi-agent-orchestrator -l
```

Installing the npm package by itself does not register the extension with Pi.
Use `pi install`, or configure your Pi host explicitly to load the built
`dist/index.js` entry point.

## Quick start

After installation, start a Pi session and type `/agents` to open the
dashboard.

- Press `t` for the resource top view, `?` for help, `z` for daemon schedules,
  and `/perf` for performance metrics.
- Navigate the agent list with `j`/`k` or the arrow keys.
- Create a custom agent in `.pi/agents/my-agent.md`; see
  [Custom Agents](docs/custom-agents.md).

## Keyboard cheatsheet

| Key | Action |
| --- | --- |
| `j`/`k` or arrows | Navigate agent list |
| `t` | Toggle top/resource view |
| `z` | Toggle schedule view |
| `?` | Help overlay |
| `/perf` | Performance metrics |
| `Shift+K` | Kill selected agent(s) |
| `Space` | Multi-select |
| `g`/`G` | Jump to first/last |
| `Esc`/`q` | Close overlay or dashboard |

## Built-in agent types

| Type | Mode | Use when |
| --- | --- | --- |
| Explore | read-only | Parallel codebase audit and read/grep sweeps |
| Plan | read-only | Architecture pass before multi-file edits |
| Analysis | read-only + `ctx_*` | Sandboxed data/compute with `@onlinechef/context-mode` |
| general-purpose | full tools | Bounded implementation after Plan/Explore |

See [Custom Agents](docs/custom-agents.md) for the frontmatter schema.

## Features

- **Interactive TUI Dashboard** — six views with vim hotkeys.
- **Sub-agent System** — specialized agents with permission inheritance and
  partition filtering.
- **Prompt Compression** — minimal, balanced, and aggressive policies with
  per-agent overrides.
- **Cron Scheduling** — persistent recurring jobs with a daemon schedule view.
- **Handoff Protocol** — structured handoffs for chain-of-agents workflows.
- **Custom Agents** — Markdown frontmatter definitions in `.pi/agents/*.md`.
- **Swarm Coordination** — dynamic join/leave with real-time status.
- **Cross-extension RPC** — peer-extension integration.
